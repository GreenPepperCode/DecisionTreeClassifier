# Projet Decision Tree Classifier avec Iris Dataset

## Description
Ce projet vise à explorer l'utilisation d'un modèle de Machine Learning supervisé, le Decision Tree Classifier, pour la classification d'espèces de fleurs d'Iris. Le dataset Iris contient 150 exemples catégorisant trois espèces d'Iris (Iris Setosa, Iris Versicolor, et Iris Virginica), chacun décrit par quatre caractéristiques (longueur et largeur des sépales, longueur et largeur des pétales). Ce projet se concentre sur la compréhension et l'application des arbres de décision, ainsi que sur l'étude de l'importance des hyperparamètres, en particulier la profondeur de l'arbre.

## Caractéristiques du Projet
- **Algorithme Utilisé** : Decision Tree Classifier
- **Dataset** : Iris Dataset
- **Hyperparamètres Clés** : Profondeur de l'arbre (`max_depth`)
- **Métriques d'Évaluation** : Accuracy sur les ensembles d'entraînement et de test

## Résultats et Conclusion
Les résultats montrent que le Decision Tree Classifier avec une profondeur d'arbre variant de 3 à 10 atteint une haute précision sur le dataset Iris. Une profondeur d'arbre de 3 s'est avérée suffisante pour obtenir une prédiction précise sans surapprentissage.
